PROJECT LUNA

Minecraft Bedrock

Warning: This addon is still in development, but already stable. Report bugs if found.

Features

Vanilla parity: Healing is now equal to Java Edition.

Soul system: Killing mobs gives souls (unlimited for now).

/unlockskill: Opens skill UI, cost = 100 souls per upgrade.

Passive skills:

  • Dash (cooldown 4s → 3s)
  • Health bonus

Entities

Soul Trader: trades souls.

Minotaur: summonable creature.

Glacial Cuboid: tamable, drops crafting material.

Weapons

Velgryn: slow but high damage.

Solaris: fast attack speed.

Ironhorn Cleaver: epic weapon, no skills.

Armor

Glacial Set: gives speed, stun, and slow effects.

Structures

Mino Maze: summon Minotaur using magic dust.

Soul Trader House: spawns in multiple biomes.

Addon will be updated regularly.
